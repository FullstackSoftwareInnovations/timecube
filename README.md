# Install

```bash
npm install timecube
```

# Usage

```js
const timecube = require('timecube')

function PraiseFourCornerTime () {
  console.log(timecube.generate(1, timecube.max))
}
```

# Parameters
### start: number (optional) 
default - 1
### stop: number (optional) 
default - timecube.max
